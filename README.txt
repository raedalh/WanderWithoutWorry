# README for Wander Without Worry

## Project Title
Wander Without Worry

## Description
Wander Without Worry is a personal blog focused on sharing information and experiences related to nature, camping, and outdoor adventures. The goal is to help others enjoy these activities affordably while providing tested information.

## Technologies Used
- HTML
- CSS
- Google Fonts

## Features
- Responsive design
- Blog posts with images
- Subscribe section for newsletters
- Related articles section to enhance user engagement
- Clear navigation links

## File Structure
- **index.html**: Main landing page of the blog.
- **about.html**: About page detailing the blog's purpose and content.
- **blog-post.html**: Detailed post on "DIY Fire: A Step-by-Step Guide."
- **blog-post2.html**: Detailed post on "Exploring the Mountain Range."
- **assets/**: Directory containing images used in the blog.
- **styles/**: Directory containing CSS files for styling the blog.
    - **main.css**: Main styling file for the website.
    - **typography.css**: Typography styling for headers and paragraphs.
    - **colors.css**: Color styling for various elements.

## Usage
To view the blog, open the `index.html` file in a web browser. For a better experience, use a live server.
